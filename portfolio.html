    Viewing Party is a program that lets users collect. modify, and organize movie data, such as: watched movie lists, favorite movie lists, friends watched movie lists, and more!

  • Task List API

    Task List is a web API that helps a user stay organized by making it easy to create read, update, and delete tasks among other features.

  • Adagrams

    Adagrams is a fun game in which a player is given a set of random letters and has to use those letters to create a word. Sunny worked on this project with Marie Keefer.

  • Swap Meet

    Swap Meet is a program that helps users organize swap meets by tracking a vendors inventory, wanted items, and swapped items among other features. Sunny worked on this project with Samiya Ismail.
